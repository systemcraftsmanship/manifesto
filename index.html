--- layout: default title: "Manifesto for System Craftsmanship" ---

As aspiring System Crafters we are raising the bar of professional system and infrastructure development even further, by extending Software Craftsmanship movement, to create more value in the DevOps ecosystem. Through this work we have come to value:

Not only well-crafted software, but also well-crafted infrastructure

Not only responding to change, but also steadily adding value

Not only individuals and interactions, but also a community of professionals

Not only customer collaboration, but also productive partnerships

That is, in pursuit of the items that the Software Craftsmanship Manifesto has, we have found the extension of the first item indispensible.
